# 续声

续声是一个本地运行的抖音私信辅助控制台，用来管理联系人画像、AI 自动回复、手动拟回复和续火花任务。应用基于 Electron 打包，数据和 API Key 都保存在本机，适合需要在电脑端统一处理抖音私信、保持聊天节奏和补发火花消息的场景。

## 主要功能

- 抖音登录与联系人同步：在本机打开抖音登录窗口，读取联系人列表和最近消息预览。
- AI 自动回复：对方发来新消息后按“一问一答”策略生成并发送回复，可按联系人单独开关。
- 手动拟回复：输入或粘贴对方消息，实时查看学习、生成、发送状态，再确认发送。
- 联系人画像：为不同联系人配置称呼、关系、语气偏向、回复禁区、注意事项和语气样例。
- 续火花任务：支持文字、表情包、文字 + 表情包三种续火花方式，并可立即发送或定时补发。
- 回复策略：配置关键词固定话术，命中后优先使用固定回复，未命中时再调用 AI。
- 模型设置：支持 OpenAI 兼容接口，可添加多个模型、测试连接，并手动选择默认模型。
- 运行记录：查看 AI 调用、自动发送、续火花和失败原因。
- 外观与设置：主题、字体、强调色、默认 AI 语气、通知、启动行为统一在设置页管理。

## 安装使用

当前 Windows 安装包请在 GitHub Releases 中下载：

```text
续声_安装包_v0.1.0.exe
```

下载并双击安装后启动续声，先登录抖音，再进入“模型设置”添加一个 OpenAI 兼容接口。配置完成后可以同步联系人、生成手动回复，或开启 AI 自动回复和续火花任务。

## 本地开发与测试

源码主要在 `app-source/`：

```text
app-source/dist/       前端页面与样式
app-source/electron/   Electron 主进程、抖音自动化、AI 服务和本地存储
app-source/test/       Node 测试
```

运行测试：

```bash
cd app-source
npm test
```

重新打包应用 ASAR：

```bash
npx --yes @electron/asar pack app-source resources/app.asar
```

重新生成安装包需要 Inno Setup 6：

```powershell
& "C:\Users\22816\AppData\Local\Programs\Inno Setup 6\ISCC.exe" "setup.iss"
```

## 隐私说明

- 应用在本机运行，配置和运行记录默认保存在 Electron 的用户数据目录。
- API Key 通过 Electron `safeStorage` 加密保存。
- AI 回复会根据你配置的模型接口发送必要的联系人画像、输入消息和上下文。
- 请在符合平台规则和对方预期的前提下使用自动回复与续火花功能。
